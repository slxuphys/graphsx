Custom Shapes

<Shape id="Pair">...</Shape>
<Pair id="P1" />
Address paths use IDs: P1.in is a public port, and P1.left.in is a port on a child shape.

Repeat

<Repeat count={4} as="i" step={[80, 0]}>...</Repeat>
id="box{i}"
from="box{i}.out" to="box{i+1}.in"
Nested repeats can use different names, such as row and col, for grids.
